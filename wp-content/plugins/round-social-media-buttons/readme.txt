Provides a responsive social media widget that displays up to eight different social media websites.

== Description ==

This plugin creates a responsive widget that displays up to eight different social media websites. (Not all 8 are required)

1. Facebook
2. Twitter
3. Google+
4. YouTube
5. LinkedIn
6. Instagram
7. Pinterest
8. Tumblr


== Installation ==

= Upload Manually =

1.Download and unzip the plugin.
2.Upload the round_social_media_buttons to the 'wp-content/plugins/' directory.
3.Go to the Plugins admin page and activate the plugin.

= Install via Admin Area =

1.In the admin area, go to Plugins > Add New and search for 'Round Social Media Buttons'.
2.Click install then click activate.

= To Use the Plugin =

1.Enter the URL to any of the eight social media websites.
2.Click Save.


== Frequently Asked Questions ==

= Are there more social media websites that can be added? =

In future updates, more social media sites will be added depending upon the amount of requests. 

= Can I change the color of the buttons? =

Yes, in your stylesheet change the background color for the following classes:
1.<code>.social_media_buttons_frontend li a.icon-facebook</code>
2.<code>.social_media_buttons_frontend li a.icon-twitter-alt</code>
3.<code>.social_media_buttons_frontend li a.icon-google-plus</code>
4.<code>.social_media_buttons_frontend li a.icon-youtube-alt</code>
5.<code>.social_media_buttons_frontend li a.icon-linkedin-alt</code>
6.<code>.social_media_buttons_frontend li a.icon-instagram</code>
7.<code>.social_media_buttons_frontend li a.icon-pinterest</code>
8.<code>.social_media_buttons_frontend li a.icon-tumblr</code>
