== Description ==

This plugin is used for those who want to show some related links in posts/pages, and those related links is external (outside the blog site).

For example, your blog site is http://blog.your.com, and you want to show the link http://www.abcd.com as a related link in your blog. You can use this plugin to get this done.

This plugin is based on the "keyword matching", when you add a external link, you need to specify a keyword for it. And if your post/page have the keyword, the link will be displayed, if not, it won't be displayed.

== Installation ==

To install this plugin, it is very easy:

e.g.

1. Upload `Ext-Relevant` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php Ext_related_links(5); ?>`, where 5 is the number of links you want to display, in your templates where you want the links to appear.
